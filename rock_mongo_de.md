# Einleitung #

**[RockMongo](rock_mongo.md)** ist ein Administrations-Werkzeug für MongoDB. Geschrieben in PHP5.

Hauptfeatures:
  * Open source mit New BSD Lizenz
  * Performant und einfach zu installieren
  * I18N (Chinesisch (simplifiziert und traditionell), Deutsch, Englisch, Französisch, Japanisch, brasilianisches Portugugisisch)
  * System
    * Mehrere Hosts mit mehreren Administratoren pro Host
    * Einfacher Wechsel zwischen den Hosts
    * Passwortschutz
  * Server
    * Serverinfo (web server, PHP, directives in PHP.ini ...)
    * Status
    * Datenbankübersicht
    * Replikationsstatus
  * Datenbank
    * Abfragen, erstellen, löschen, reparieren
    * Ausführung Befehle und JavaScript-Codes aus
    * Statistiken
    * Profiling
    * Benutzerverwaltung
    * Datentransfer (Klonen)
    * Export/Import
  * Sammlungen
    * Erweiterte Abfragen- und Erklärungswerkzeuge
    * Lesen, einfügen, modifizieren, duplizieren und löschen einzelner Zeilen
    * Abfrage, Erstellung und Löschung von Indizes
    * Validierung von Sammlungen
    * Leeren von Sammlungen
    * Regelbasierte Massenmodifizierung und -löschung von Zeilen
    * Statistiken
    * Umbenennung
    * Modifikation von Eigenschaften (Größenbeschränkungen, capped, max, usw.)
    * Datentransfer (Klonen)
    * Export/Import
  * Felder
    * Modifizieren, löschen, leeren
  * GridFS
    * Chunks ansehen
    * Dateien downloaden
  * mehr Features sind geplant und bereits in Entwicklung!

# Installation #

  1. Das Paket [downloaden](http://rockmongo.com/downloads)
  1. Paket im Wurzelverzeichnis der Website entpacken
  1. Die Datei config.php mit einem beliebigen Texteditor öffnen und die Parameter anpassen
  1. Nun die index.php im Browser öffnen (z.B. http://localhost/rockmongo/index.php)...
  1. ... und mit dem eingestellten Admin-Usernamen und -passwort einloggen (Standardeinstellungen: Benutzername: admin, Passwort: admin)
  1. spiel mit deiner MongoDB :-)

Tutorials:
  * [Instalando RockMongo – Interfaz de Administracion web para MongoDB](http://blog.jam.net.ve/2011/01/11/instalando-rockmongo-interfaz-de-administracion-web-para-mongodb/)
